# WhatToEatForDinner
What to eat for dinner


## 安裝
```bash
# (npm 安裝沒問題，但是 npm 要升級
# nvs 每個版本裝的套件都是獨立的，所以切到最低版本 18，要再安裝一次 gemini)
# 目前用 Node.js 20
npm install -g @google/gemini-cli
gemini
```

## 新增 API KEY
```bash
# GEMINI API KEY 新增 Powershell 在 $profile
$Env:GEMINI_API_KEY = 'A-------------------------------------k'
```

## 執行
```bash
gemini
```
